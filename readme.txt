This is a text editor.

It opens file C:/shared/text/RANDOM.txt by default,
create it before running it.

Controls:

q - quit
ikjl - up down left right
o m - scroll up/down by 3 lines
; - open filepath under cursor
p n - go to previous/next file
enter - to start typing
esc - to stop typing
z - undo
shift-z - redo
x - cut
c - copy
v - paste
e r - previous/next word
E R - start/end of line
(press E repeatedly to switch between start of text
and actual start of line)
/ - search
. - search down
, - search up

there are probably more, read source code
